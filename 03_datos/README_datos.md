# Fuente de datos

Los datos provienen del **Ministerio de Energía y Minas del Perú (MINEM)**.

- URL oficial: https://www.minem.gob.pe
- Sección: Minería → Estadísticas → Accidentes
- Período: 2002 – 2021
- Registros: 901 accidentes mortales
- Formato original: Excel / CSV

> Por restricciones de licencia los datos crudos no se incluyen en este repositorio.
> Descárgalos directamente del portal MINEM usando el link anterior.
